我的电脑环境备份
---

当前电脑为mac笔记本
主要的操作以vim为主

使用yabai + alactritty + zsh + ohmyzsh + tmux + neovim 的配置完成整体布局。
配合spacebar来查看界面的窗口显示（需要隐藏mac上方的控制栏）。

* control：来完成**tmux**下的界面控制与管理。
* option：主要以控制**yabai**来完成屏幕切换、调整的操作。
* command：主要完成uTool的软件控制。
