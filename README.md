我的电脑环境备份
---

当前电脑为mac笔记本
主要的操作以vim为主

使用yabai + skhd + alactritty + zsh + ohmyzsh + tmux + neovim 的配置完成整体布局。
配合spacebar来查看界面的窗口显示（需要隐藏mac上方的控制栏）。
uTool：软件、文件、小插件配合使用

* control：来完成**tmux**下的界面控制与管理。
* option：主要以控制**yabai/skhd**来完成屏幕切换、调整的操作。
* command：主要完成uTool的软件控制。

快捷键简介：
   快捷键  | 效果
:--------:|:-----:
 ⌥ + c    | 新建窗口
 ⌥ + n/p  | 向下/上一个屏幕切换
 ⌥ + hjkl | 焦点切换
 ⌥ + f/F  | 全屏/真全凭，再次点击回复
 ⌥ + 0    | 平衡界面布局
 ⌥ + 1/2/3| 切屏到指定位置
 ⌥ + x    | 关闭窗口
 ⌥ + wasd | 增大窗口（只能增大，不能缩小）
 ⌥ + ⇧ + wasd | 缩小窗口（只能缩小，不能增大）
 ⌥ + f/F  | 全屏/真全凭，再次点击回复
 ⌥ + t    | 不受**yabai**控制，再次点击恢复
 ⌥ + o    | 缩略图，方便移动，再次点击恢复
